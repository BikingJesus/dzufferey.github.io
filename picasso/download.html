PICASSO: a PI-CAlculus-based Static SOftware analyzer

Download

Picasso is available in different versions:

  • Compiled version (jar).

    Version that runs out of the shelf (modulo external tools). See the manual section of this page to learn how to run Picasso.

  • Picasso repository at https://github.com/dzufferey/picasso

  • The sources of the core components (graph rewriting and backend).

    The archive contains all the files to compile and run Picasso on graph rewriting systems.

  • The sources of the basic frontend.

    The archive contains a frontend to run Picasso on pi-calculus processes (with human friendly syntax). The core components are required to use the frontend.

  • The sources of the scala compiler plug-in.

    The archive contains a frontend to run Picasso on scala programs (small subset of the language). The core components are required to use the frontend.

Currently, the scala plug-in is on hold and we do not recommend using it. Hopefully, we will find time to rewrite it at some point in the future.
